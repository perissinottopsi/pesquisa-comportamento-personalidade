# pesquisa-comportamento-personalidade
Pesquisa acadêmica (UNIP-Goiânia) sobre traços da Tétrade Sombria e comportamentos contemporâneos. Coleta dados sobre uso de redes sociais, True Crime e atividade física, avaliando ansiedade, autoestima, autoeficácia e satisfação com a vida. Participação anônima, voluntária, +18 anos.
